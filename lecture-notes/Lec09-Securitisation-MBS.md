## 1. 什么是证券化？Why Securitisation?

**证券化（Securitisation）的本质：** 把不流动的资产（如抵押贷款）打包成可交易的证券，出售给更广泛的投资者，从而为原始贷款方补充资金。

### 类比理解

```
传统模式（无证券化）：
  你去银行借款 → 银行用储户存款发放贷款 → 贷款锁定在银行资产负债表上 30 年
  问题：银行资金被"锁住"，无法再发放新贷款

证券化模式：
  你去银行借款 → 银行发放贷款 → 把贷款卖给 GSE → 
  GSE 打包成证券卖给投资者 → 银行拿回资金，继续发放新贷款
  好处：银行资金快速"解锁"，可以帮助更多人买房
```

## 2. 二级市场起源 Why Did Secondary Market Emerge?

### 大萧条前的美国抵押贷款（问题重重）

**大萧条（Great Depression，1929–1939）之前，美国抵押贷款条件极其苛刻：**

```
首付要求：40–50%（普通家庭根本付不起！）
贷款类型：纯利息贷款（IO），到期不还本金
期限极短：3–5 年，到期后需要再融资（Refinance）
问题：到期时银行不续贷，借款人被迫还全款 → 走投无路 → 违约
```

**大萧条期间的灾难：**
- 1/3 的美国银行倒闭
- 1/3 的人口失业
- **50% 的抵押贷款违约**！（是正常年份的数十倍）
- 私人贷款机构因惧怕损失，拒绝提供新贷款
- 美国大约 45% 的人口是租房者（买房非常困难）

### 政府干预：二级市场的建立

**为什么二级市场能解决问题？**

1. **补充资金来源（Replenish Funds）：**
   - 银行把已发放的贷款出售给 GSE（政府支持企业）
   - 拿到现金后再发放新贷款 → 资金循环利用
   - 解决了银行"弹药不足"的问题

2. **风险转移（Shifting of Risks）：**
   - 贷款风险从银行转移到最终的证券投资者
   - 银行不用再担心 30 年后的违约风险

3. **专业化分工（Returns to Specialisation）：**
   - 银行专注于：评估借款人信用、发放贷款（Origination）
   - GSE 专注于：打包证券、信用担保
   - 投资者专注于：购买符合自己风险偏好的 MBS

## 3. 美国 GSE（政府支持企业）发展时间线

| 年份 | 事件 | 意义 |
|------|------|------|
| **1938** | FNMA（Federal National Mortgage Association）成立，即"Fannie Mae" | 大萧条后政府创立的二级市场机构，增强抵押贷款市场流动性 |
| **1968** | FNMA 拆分：Fannie Mae 私有化；Ginnie Mae（Ginnie Mae，GN）独立 | Fannie 变成"私人公司但有隐性政府背书"；Ginnie 处理政府担保的贷款 |
| **1970** | Freddie Mac（Federal Home Loan Mortgage Corporation）成立 | 为 Fannie 提供竞争，专注于为传统（未有政府担保的）抵押贷款提供二级市场 |
| **2008** | Fannie & Freddie 被美国政府接管（国有化） | 次贷危机，两房资产质量崩溃，政府救助 |
| **2013** | 两房"表现改善" | 危机后恢复，但仍在政府托管中 |
| **2026** | ？ | 两房未来独立或改革，政策仍不明朗 |

### Fannie 和 Freddie 最初做什么？

**Fannie Mae：**
- 原始职能：增强联邦政府担保或保险贷款的二级市场交易
- 私有化后：仍有美国财政部可购买高达 22.5 亿美元票据的"隐性背书"
- 这意味着：Fannie 出问题，政府会出手 → Fannie 可以低成本融资

**Freddie Mac：**
- 为传统（非政府担保的）抵押贷款提供二级市场和流动性
- 通过竞争向 Fannie 施压，防止垄断，保持市场效率

## 4. MBS 运作机制 How MBS Works ⭐

### 今天典型美国抵押贷款的生命周期

```
阶段 1：一级市场（Primary Market）
  消费者（借款人） ← 贷款 ← 贷款机构（银行/信用合作社）
  抵押品（房产）作为担保

阶段 2：贷款机构的选择
  A. 留在资产负债表上（收取利率差）
  B. 出售给 GSE/发行人 → 换取现金 + 收取服务费（Servicing Fee）
     服务内容：收取还款、管理账户、处理违约

阶段 3：二级市场（Secondary Market）
  GSE 从贷款机构购入贷款 → 打包成 MBS → 出售给投资者
  GSE 收取担保费（G-Fee）承担信用风险
  投资者购买 MBS → 获得抵押贷款的现金流权利
```

### 两种 GSE 运作模式

**现金项目（Cash Programme）：**
```
GSE 直接购买贷款（付现金给银行）→
→ 选择一：持有贷款赚利差（收贷款利息 - 付债券利息）
→ 选择二：打包成 MBS 出售给投资者（GSE 担保信用）
```

**互换项目（Swap Programme，最常见）：**
```
银行把贷款"互换"给 GSE →
GSE 不付现金，而是返给银行同等价值的 MBS 证券 →
银行然后把 MBS 卖给投资者，获得现金
GSE 收取担保费（G-Fee）并承担信用风险

变体：GSE 也可以把 MBS 进一步结构化为 CMO（担保抵押债券）
```

> **GSE 的盈利来源：** 担保费（G-Fee）—— 即承担信用风险的报酬；利差（如果持有贷款）

## 5. MBS 发行方如何融资？

GSE 发行的主要债务工具：
- **长期、中期、短期票据（Notes）：** 满足不同投资者的期限需求
- **基准票据（Benchmark/Reference Notes）：**
  - 模仿美国国债的特征（流动性强，大额发行）
  - 通常利率高出国债 25 基点（0.25%）
  - 高度流动性 → 机构投资者喜爱
- **外币债券：** 向全球投资者融资

## 6. RMBS 的主要风险 Key Risks of MBS ⭐

证券发行方（GSE 或私人机构）必须管理以下五大风险：

### 风险 1：利率风险（Interest Rate Risk）

**问题：** 发行方从抵押贷款收到利息，同时发行债券支付利息。如果利率突变，两边的"利率"错位，发行方可能亏损。

**管理工具：久期（Duration）**

```
久期 = 现金流现值的加权平均到期时间

Duration = (1 × PV(CF₁) + 2 × PV(CF₂) + ... + N × PV(CFₙ)) / 总价值

其中：
  t = 第 t 期
  PV(CFₜ) = 第 t 期现金流的折现值
  总价值 = 所有现金流的折现值之和
```

**直觉：** 久期越长 = 现金流越"后置"= 对利率变化越敏感（就像长期债券比短期债券对利率更敏感）

**久期缺口（Duration Gap）：**
```
= 资产（抵押贷款）久期 - 负债（发行债券）久期

理想状态：久期缺口 ≈ 0（资产和负债"久期匹配"）
→ 利率变化时，资产价值变化 ≈ 负债价值变化，净值稳定

若久期缺口过大：利率突变时，资产和负债价值变化不对等 → 净值大幅波动
```

**如何管理久期缺口？**
- 买进或卖出适当期限的抵押贷款（调整资产端）
- 发行或回购适当期限的债券（调整负债端）
- 使用**可赎回债券（Callable Debt）：** 利率下降时，可以提前赎回旧债重新发行 → 帮助管理提前还款风险

### 风险 2：信用风险/违约风险（Credit Risk / Default Risk）

**评估维度：** 贷款机构评估借款人的"3C"：
```
1. Capacity（还款能力）：收入水平、债务收入比（DTI）、就业稳定性
2. Credit History（信用历史）：FICO 信用评分（300–850分，越高越好）
3. Collateral（抵押品）：房产价值、LTV 比率（越低越安全）
```

**现代工具：**
- FICO 评分模型：自动化信用评估，减少主观判断
- 房价风险模型：预测不同地区房价下跌概率，评估抵押品未来价值

**信用增强（Credit Enhancements）：**
- 过度抵押（Over-Collateralisation）：MBS 背后的贷款总额 > MBS 发行额
- 保险（Insurance）：PMI（Private Mortgage Insurance）等

### 风险 3：提前还款风险（Prepayment Risk）

**为什么提前还款是风险？**
```
利率下降 → 借款人以更低利率再融资（Refinancing Boom）
         → 提前还款，把本金还给 MBS 发行方
         → 发行方拿回本金后，只能以更低的新利率再投资
         → 损失：再投资收益率 < 原来的贷款利率
         = "再投资风险（Reinvestment Risk）"
```

**提前还款让预测 MBS 现金流变得极难：**
- 不知道借款人何时还款（利率下降就大量还，利率上升就坚持不还）
- 这种"非线性"行为使 MBS 定价复杂

**提前还款测量指标：**

| 指标 | 全称 | 含义 |
|------|------|------|
| **CPR** | Conditional Prepayment Rate | 年化提前还款率：一年内还掉多少比例的贷款 |
| **PSA** | Public Securities Association Standard | 标准提前还款速度曲线：前30个月CPR线性增至6%/年，之后维持6% |
| **SMM** | Single Monthly Mortality | 月化提前还款率 |

```
PSA 说明：
100% PSA = 标准速度（30个月达到 6% CPR）
200% PSA = 两倍标准速度（提前还款更快）
50% PSA  = 一半标准速度（提前还款更慢）

用于情景分析：若 PSA = 200%，MBS 多快会被还清？价格怎么变？
```

### 风险 4：政治风险（Political Risk）

- 政府更改贷款规定、税法、住房补贴政策等，影响 MBS 价值
- 如：政府限制抵押贷款利率 → 贷款供应减少 → MBS 规模萎缩

### 风险 5：流动性风险（Liquidity Risk）

- 市场恐慌时，MBS 可能没有买家（即使质量好也卖不出去）
- GSE 发行量大，流动性好；私人 MBS 流动性较差

## 7. MBS 类型 Types of MBS ⭐

### 类型一：抵押担保债券（MBB，Mortgage-Backed Bond）

**特征：**
```
发行方（通常是大银行或金融机构）：
  建立一个抵押贷款池 → 发行债券
  发行方保留贷款的所有权（贷款留在资产负债表）
  通常"过度抵押"：贷款池总额 = 125%–240% × 债券发行额
  过度抵押 = 安全缓冲（即使部分贷款违约，仍能支付债券本息）

投资者：
  持有的是债券（固定票息 + 到期本金）
  不直接承担违约风险和提前还款风险（发行方兜底）
  发行方赚差价：从贷款收取利息 - 向债券持有人支付利息
```

**定价例子：**
```
条件：
  债券：面值 £10,000，票息 8%，10年期，抵押物：£300M 贷款池 vs £200M 债券
  → 150% 过度抵押

若市场要求回报率（RR）= 9%（> 票息 8%）：
  PV = 800/(1.09)^1 + ... + 800/(1.09)^10 + 10,000/(1.09)^10
     = £9,358
  → 折价出售：93.58% of par

若 RR = 7%（< 票息）→ 溢价；RR = 8% → 平价
```

---

### 类型二：过手证券（MPT，Mortgage Pass-Through Securities）

**为什么叫"过手"（Pass-Through）？**

```
还款流程：
借款人每月还款 → 贷款服务商（Servicer）收款 →
扣除服务费（~0.5%）和担保费 → "过手"传递给 MBS 投资者

关键：本金、利息、提前还款 — 全部"过手"传递给投资者
→ 投资者直接承担：提前还款风险 + 再投资风险
```

**与 MBB 的区别：**
- MBB：发行方保留贷款，向投资者支付固定债券利息
- MPT：发行方不保留贷款，把贷款现金流"透传"给投资者

**MPT 定价例题：**
```
条件：
  贷款池：£1,000,000，10% 固定利率，10年期
  MPT 票息（Coupon）：9.5%（差0.5% = 服务费）
  40份证券 → 每份面值 = 1,000,000/40 = £25,000

若市场 RR = 9.5%（= 票息）：
  → 平价出售，每份 £25,000

若 RR = 8.5%（< 票息）：
  → 溢价，每份 £26,130

若 RR = 10.5%（> 票息）：
  → 折价，每份 £23,944
```

### 类型三：担保抵押债券（CMO，Collateralized Mortgage Obligations）⭐⭐

**CMO 解决什么问题？**

MPT 的问题：所有投资者平等承担提前还款风险，没有选择余地。但不同投资者有不同偏好：
- 保险公司需要**长期**、**稳定**的现金流
- 货币市场基金需要**短期**资产

**CMO 的解决方案：** 把贷款池的现金流**切割**成不同风险/期限的"分层"（Tranches），投资者选择适合自己的分层。

**CMO vs MPT 对比：**

| | MPT | CMO |
|--|-----|-----|
| 所有权 | 投资者共同持有贷款池 | 发行方保留贷款池，发行债券 |
| 发行方权益 | 无权益 | 有权益（过度抵押部分） |
| 分层 | 无，所有人平等 | 有多个分层（Tranches），不同优先级 |
| 提前还款风险 | 所有投资者平等承担 | 按分层优先级分配 |

---

### CMO 机制详解（数字案例）

**例：** CMO 发行额 £72M，贷款池 £75M（差 £3M = 过度抵押 / 发行方股权）

贷款池：10年期，11% 利率，年度 CPM

**三个分层：**
```
A 档（Tranche A）：£27M，票息 9.25%，预计期限 2–5年（短期）
B 档（Tranche B）：£15M，票息 10.0%，预计期限 4–7年（中期）
Z 档（Tranche Z）：£30M，票息 11.0%，预计期限 6–10年（长期）
```

**加权平均利率：** (27×9.25 + 15×10 + 30×11) / 72 = 10.14%

→ 发行方收 11%（贷款池收益），付 10.14%（各分层票息），赚差价！

**还款顺序（Sequential Payout，依次偿还）：**

```
A 档（优先级最高，最早还清）：
  收到：自己的票息（9.25% × 余额）
      + Z档的"应付利息"（11% × Z余额）—— Z不领，全部优先给A还本
  优先收回本金和提前还款
  
B 档（次优先）：
  A 还清前：只收 B 的票息（10% × B余额）
  A 还清后：开始收回本金（类似A档的机制）
  
Z 档（最后还清）：
  A 和 B 还清前：什么都不收
  自己的利息不领，"积累计息"（11% × Z余额 复利积累）
  A + B 还清后：才开始收利息 + 本金
```

**为什么这样设计？**
- Z 档的利息积累（不拿）→ 把这笔钱给 A 档加速还本
- 结果：A 档比在普通 MPT 中更早还清（期限更短，风险更低）
- A 档投资者（如货币市场基金）得到了稳定的**短期**资产！

**分层现金流表（关键数据）：**

| 年 | 贷款池余额 | 总还款 | A档余额 | B档余额 | Z档余额(累积) |
|----|-----------|--------|--------|--------|------------|
| 0 | 75.000 | — | 27.000 | 15.000 | 30.000 |
| 1 | 70.514 | 12.735 | 19.214 | 15.000 | 33.300 |
| 2 | 65.536 | 12.735 | 10.573 | 15.000 | 36.963 |
| 3 | 60.010 | 12.735 | 0.981 | 15.000 | 41.028 |
| 4 | 53.876 | 12.735 | 0 | 5.334 | 45.542 |
| 5 | 47.067 | 12.735 | 0 | 0 | 44.067(开始减少) |
| ... | ... | ... | 0 | 0 | 继续偿还 |

**残余股权（Residual Equity CF）：**
- 发行方投入 £3M 过度抵押（年 0 负现金流）
- 每年收到贷款池总还款 - 支付给 A+B+Z 的款项 = 残差
- 最终 IRR = 20.19%（发行方的股权回报）

## 8. MBS 的收益分配 Who Benefits?

### 对贷款机构（银行）的好处
```
1. 资产变流动：贷款从资产负债表消失，换成现金
2. 补充资金：可以用新资金发放更多贷款
3. 风险转移：违约风险、利率风险转移给投资者/GSE
4. 专业分工：银行专注于信用评估，不必长期管理风险
```

### 对投资者的好处
```
1. 新型投资产品：间接投资房地产市场
2. 多样化：可以选择不同风险/期限的分层
3. 适合不同风险偏好：机构投资者可选择 AAA 级 Senior Tranche
```

### 对借款人（房主）的好处
```
1. 更低的利率：二级市场降低了贷款成本
   估计：MBS 使抵押贷款利率降低约 50 基点（0.5%）
   长期（30年）节省：普通家庭约 $10,000（稍旧的估算）
2. 更广泛的贷款供应：银行资金循环加快，更多人能借到钱
```

## 9. 问题与担忧 Concerns

### 隐性政府担保的道德风险

```
Fannie & Freddie 有隐性政府担保
→ 可以以比私人机构更低的利率融资
→ 以此获得竞争优势，垄断市场
→ "太大而不能倒"（Too Big to Fail）

2008年证明：确实太大不能倒 → 政府救助
结果：私人收益，公共风险（纳税人兜底）
```

### 次级贷款与 2008 年金融危机 Subprime Mortgages & GFC

**什么是次级贷款（Subprime Mortgage）？**
```
向还款能力存疑的借款人发放的贷款：
- 低 FICO 信用评分
- 高 DTI（债务收入比，55%+）和高 LTV（85%+）
- 差文件记录（Alt-A，如"无收入核实"贷款）
```

**典型次级贷款产品——"2/28 ARM"：**
```
最初2年：低"诱惑利率"（Teaser Rate，如 8%）
2年后"重置"：LIBOR + 高加成（如 LIBOR + 6% = 12–14%）
→ 月供大幅跳升（可能 +50–100%！）
→ 借款人无力支付，被迫再融资或违约

设计逻辑（当时的想法）：
  2年内房价上涨 → LTV 下降 → 再融资还清 2/28，换更好贷款
  = 完全依赖房价持续上涨！
```

**为什么爆发危机？**
```
2004–2006：房价飙涨 → 证券化降低了发放次级贷款的成本 → 
  发放量暴增 → 评级机构评级过高（误认为房价不会同时全国下跌）

2007：利率上升 + 房价停止上涨/下跌 → 
  借款人无法再融资 → 开始违约 → 
  大量法拍房涌入市场 → 进一步压低房价 → 更多违约...
  恶性循环！

结果：次级 MBS 价值暴跌 → 持有这些证券的银行/机构大量亏损 →
  2008年全球金融危机
```

**风险分散还是风险隐藏？**
```
证券化的初衷：将风险分散给能承受的投资者
实际结果：风险被"隐藏"在复杂的金融产品中
  次级MBS → 打包进CDO（担保债务凭证）→ 
  再打包进CDO的CDO（"CDO平方"）→ 
  风险变得不透明
  
最终：没有人知道自己手里的CDO里有多少次级贷款暴露
"中国盒子"游戏：一层套一层，信息完全丢失
当房价下跌，整个体系崩溃
```

## 10. 次级证券的特殊结构 Subprime Securities Features

**次级 MBS（SS，Subprime Securities）的特别之处：**

```
1. 超额利差（Excess Spread, XS）：
   收到的利息 > 向债券持有人支付的利息 → XS 建立"超额抵押"（OC）
   
2. 超额抵押（Over-Collateralisation, OC）目标：
   SS 有 OC 目标；只有达到目标后，XS 才能分给剩余持有人
   
3. 降档日（Step-Down Date）：
   此前：优先级最高的证券获得 100% 的提前还款本金
   此后：按新的优先级分配
   
4. 触发条件（Performance Triggers）：
   如果逾期率、损失率超标 → 停止向中间/劣后债券支付 → 保护优先级
   
5. 可用资金上限（Available Fund Cap, AFC）：
   因为底层是 ARM，实际利率可能低于票息 → 限制支付，防止超付
```

**为什么次级证券定价那么难？**
```
几乎所有收益都依赖借款人"正常再融资"
而再融资 = 依赖房价上涨 + LTV 降低
一旦房价下跌 → 再融资停止 → 各种"触发条件"被激活 → 
原本看似"安全"的优先级债券也无法独善其身
评级机构对此准备不足
```

## 考试高频考点 Exam Focus Points

> **历年真题分析 Based on Past Exams 2013–2025**

> ⚠️ **考试趋势**：MBS/证券化 2013–2016 年为核心必考内容；2017–2021 年偶尔出现于理论题；2022 年后几乎不再考，ESG 和开发融资取而代之

1. **MBS/证券化机制（早期必考）** ★★★★ — 2013, 2014, 2016
   - Cash Programme vs Swap Programme 区别（GSE 如何分配信用风险）
   - 风险如何在：贷款机构 → GSE → MBS 投资者 之间传递

2. **CMO vs Pass-Through Securities** ★★★ — 2013, 2014, 2016, 2018（resit）
   - Pass-Through：所有投资者平等承担提前还款风险
   - CMO 分层（Senior/Mezzanine/Junior）：高级层最先收到还款，最后承担损失
   - 考题常要求讨论"tranching 如何改变风险分配"

3. **次级贷款与 2008 危机** ★★★ — 2013, 2014, 2018（resit）
   - 次级贷款（Subprime）特征：通常为 ARM，与房价上涨强依存
   - 为何 Subprime MBS 难定价：overcollateralization 不稳定，delinquency triggers 复杂
   - 评级机构失误：假设全国房价不会同时下跌 → 评级虚高 → 危机

4. **REITs 间接投资（2016–2022 高频）** ★★★★★ — 2013, 2014, 2015, 2016, 2017, 2018, 2019, 2020, 2022, 2023
   - 绕过直接投资三大风险：流动性差、不可分割性、信息不对称
   - 主要约束：强制分红（90%+ 应税利润）、限制再投资
   - 价值创造策略：管理改进、收购优质资产、资本结构优化
   - UK REIT 历史（2007 年启动以来）：用杠杆和 NAV 分析（2014–2017 反复考）

5. **ESG / 绿色投资（近年新重点）** ★★★ — 2022, 2023, 2024
   - 气候变化如何影响 NOI（能效、碳税、绿色溢价/棕色折扣）
   - 绿色建筑在 Pro Forma 中的体现：更低运营成本、更高租金、更低 Cap Rate

6. **投资组合构建 Portfolio Construction** ★★★ — 2019, 2020, 2021, 2023
   - 关键考量：流动性、资产特征（物业类型/地域）、物业数量（分散化）
   - 直接 vs 间接组合的差异；私有市场数据限制（Return smoothing 问题）
