## 1. 风险评估方法 Risk Assessment Methods

**为什么需要风险评估？** 投资决策不能只看 IRR 或 NPV，因为高回报必然伴随高风险。一个 IRR = 20% 的投资，如果有 30% 概率亏损 50%，它和 IRR = 15% 但风险极小的投资哪个更好？需要系统的风险评估框架。

### 方法一：尽职调查 Due Diligence

**什么是尽职调查？** 在收购前，对物业进行系统性的全面调查，确认投资风险是否符合投资目标。

**尽职调查核查清单（典型内容）：**
- 物业状况：建筑结构、机电系统、环境污染
- 租赁情况：租约条款、租户信用评级、空置历史
- 市场分析：供需状况、竞争物业、租金走势
- 法律合规：产权清晰、规划许可、无违法用途
- 财务审查：历史 NOI 准确性、运营费用合理性

> **为什么尽职调查特别重要？** 当机构（如养老金、保险公司）代他人管理投资时，必须有记录在案的系统性调查，以满足**问责制（Accountability）**要求——否则出了问题，管理人要承担个人责任。

### 方法二：IRR/PV 分解 Partitioning the IRR/PV

**核心思路：** 把总 PV 分解为两部分，分析各部分对总价值的**贡献比例**，判断投资风险集中在哪里。

```
总 PV = PV（经营现金流）+ PV（出售收益）

风险程度：出售收益 > 经营现金流
（因为出售收益发生在未来，价格更不确定）
```

**典型例子（办公楼）：**
- 经营 CF 的 PV = 总价值的 30%
- 出售收益的 PV = 总价值的 70%

**风险含义：** 如果 70% 的价值来自 10 年后才能确定的售价，这笔投资风险**很高**——如果 10 年后市场低迷，你的大部分回报可能蒸发。

> **规律**：长期高增长预期（如高 g，用 Gordon 模型估出的高售价）的投资，终值比重越大，对市场预测的依赖越高，风险越大。

### 方法三：敏感性分析 Sensitivity Analysis ⭐

**核心思路：** 给出三种情景（悲观、最可能、乐观），计算每种情景下的 IRR，然后计算期望 IRR 和风险指标。

**公式体系：**
```
期望 IRR：
E(IRR) = Pr₁ × IRR₁ + Pr₂ × IRR₂ + Pr₃ × IRR₃
（概率加权平均）

方差（Variance，衡量离散程度）：
Var = Pr₁ × (IRR₁ - E(IRR))² + Pr₂ × (IRR₂ - E(IRR))² + Pr₃ × (IRR₃ - E(IRR))²

标准差（Standard Deviation，风险大小）：
SD = √Var

变异系数（CoV，每单位回报对应的风险）：
CoV = SD / E(IRR)
```

**哪些变量可以"弹性调整"？**
- NOI（改变租金假设或空置率）
- 增长率 g（经济好坏）
- Going-out Cap Rate（退出市场状况）
- 折现率 r（利率环境变化）

**如何使用结果？**
- SD 越小 → 风险越低
- CoV 越小 → 每单位回报的风险越低（更有效率）
- 可以比较两个投资项目的 E(IRR) 和 CoV，寻找最优风险-回报组合

## 2. 财务杠杆基础 Financial Leverage Basics

**什么是财务杠杆？** 借入资金来放大投资规模，从而（可能）放大回报。

### 为什么使用债务 Why Use Debt？

1. **自有资金不足**（Lack of Equity Capital）：房地产价格高昂，纯靠股权难以完成大额投资
2. **利息税盾**（Tax Deductibility of Interest）：贷款利息可抵税，降低实际借贷成本
3. **正杠杆效应**（Positive Leverage）：若借款利率 < 物业整体回报率，借钱可放大股权回报
4. **改变风险特征**（Change Risk Profile）：有时通过债务结构调整项目风险-回报特征

### 核心术语

```
资产价值 V = 债务 D + 股权 E

LTV（Loan-to-Value Ratio，贷款价值比）= D / V
→ 表示物业价值中有多少比例是借来的
→ 例：V=100，D=80，则 LTV = 80%

LR（Leverage Ratio，杠杆率）= V / E = 1 / (1 - LTV)
→ 表示每 1 英镑股权对应多少英镑资产
→ 例：LTV=80%，则 LR = 1/(1-0.8) = 5
→ 意思：你出 20 英镑股权，控制 100 英镑资产
```

## 3. 杠杆数字案例（基于美国税法）

**情景设定：**
- 购入价：£100（建筑价值 £85，土地 £15）
- NOI 每年：£12（净营业收入）
- 贷款：£80，利率 10%，纯利息贷款（Interest Only，IO），期限 5 年
- 所得税率：28%
- 折旧：建筑按 31.5 年直线法折旧（只折旧建筑，不折旧土地！）
- 持有期：5 年，届时以 £100 卖出

```
LTV = 80/100 = 80%
LR  = 100/20 = 5（5倍杠杆）
```

### 有杠杆的现金流计算

**每年经营现金流（有杠杆）：**
```
BTCFO（税前经营现金流）= NOI - 利息 = 12 - 8 = 4

应税收入 = NOI - 利息 - 折旧
         = 12 - 8 - (85/31.5)
         = 12 - 8 - 2.698 = 1.302

ATCFO（税后经营现金流）= BTCFO - 所得税
                      = 4 - (1.302 × 28%) = 4 - 0.365 = 3.636
```

**第 5 年出售现金流（有杠杆）：**
```
BTCFS（税前出售现金流）= 售价 - 未还贷款 = 100 - 80 = 20

折旧后调整成本基础（Adjusted Basis）= 购入价 - 5年折旧
= 100 - 5 × (85/31.5) = 100 - 13.49 = 86.51

资本利得税 = (售价 - 调整成本基础) × 税率
= (100 - 86.51) × 28% = 13.49 × 28% = 3.778

ATCFS（税后出售现金流）= 20 - 3.778 = 16.222
```

**IRR 计算：**
```
有杠杆（税前）：CF0 = -20, CF1-4 = 4, CF5 = 24 → BTIRR = 20.0%
有杠杆（税后）：CF0 = -20, CF1-4 = 3.636, CF5 = 19.858 → ATIRR = 15.4%
```

### 无杠杆的现金流对比

```
无杠杆（税前）：CF0 = -100, CF1-4 = 12, CF5 = 112 → BTIRR = 12.0%
无杠杆（税后）：CF0 = -100, CF1-4 = 9.396, CF5 = 105.618 → ATIRR = 8.76%
```

**关键发现：**
- 税前：杠杆将 IRR 从 12% 提升到 20%（+8%）
- 税后：杠杆将 IRR 从 8.76% 提升到 15.4%（+6.6%）

## 4. WACC 加权平均资本成本 ⭐

### 为什么借钱能放大股权回报？— WACC 公式解释

**会计恒等式：**
```
资产 (V) = 负债 (D) + 股权 (E)
资产现金流 = 债务现金流（利息还款）+ 股权现金流
```

**定义各方回报率：**
```
rD = 债务回报率（= 贷款利率，即银行每年收到的）
rE = 股权回报率（= 股东每年的回报比例）
rP = 物业整体回报率（= NOI / V）
```

**WACC 公式（加权平均资本成本）：**
```
rP = LTV × rD + (1 - LTV) × rE

变形得到股权回报率：

rE = rP + (rP - rD) × (D/E)
```

**直觉理解：**
- 若 rP > rD（物业回报 > 借贷成本）→ 括号内为正 → rE > rP
- 借得越多（D/E 越大），rE 放大越多 → **正杠杆（Positive Leverage）**
- 若 rP < rD → 负杠杆，借钱反而损害股权回报

### 数字验证

**以上例为例（无税情景）：**
```
资产：V = 100，D = 80，E = 20
现金流：NOI = 12，利息 = 8，股权 CF = 4

rD = 8/80 = 10%
rE = 4/20 = 20%
rP = 12/100 = 12%

验证 WACC：
12% = 0.8 × 10% + 0.2 × 20% = 8% + 4% ✓

验证股权公式：
rE = 12% + (12% - 10%) × (80/20)
   = 12% + 2% × 4
   = 12% + 8% = 20% ✓
```

### 税后的正杠杆条件

税后情况下，借贷利息可以抵税，所以**税后实际借贷成本更低：**

```
税后债务成本 = rD × (1 - 税率) = 10% × (1 - 28%) = 7.2%

正杠杆税后条件：税后债务成本 < 无杠杆 ATIRR
                7.2% < 8.76% ✓ → 正杠杆有效
```

**税后股权回报估算：**
```
rE ≈ rP + (rP - rD×(1-t)) × (D/E)
   = 8.76% + (8.76% - 7.2%) × (80/20)
   = 8.76% + 1.56% × 4
   = 8.76% + 6.24% ≈ 15%
```

（注：与精确 DCF 结果 15.4% 略有差异，因为 D/E 在持有期内会变动，WACC 只是近似值）

## 5. 盈亏平衡利率 Break-Even Interest Rate (BEIR)

**BEIR 是什么？** 使得"有杠杆 ATIRR = 无杠杆 ATIRR"时的借贷利率。低于 BEIR 借贷则有正杠杆；高于 BEIR 借贷则为负杠杆。

```
BEIR = 无杠杆 ATIRR / (1 - 税率)
     = 8.76% / (1 - 0.28)
     = 8.76% / 0.72
     = 12.17%
```

**解读：** 只要贷款利率 < 12.17%，使用杠杆对股权回报有正面效果。上例中贷款利率 10% < 12.17%，所以杠杆有益。

## 6. 增量借贷成本 Incremental Cost of Debt

**应用场景：** 你已有一笔 80% LTV 的贷款，银行提供第二方案：85% LTV，但利率从 10% 升到 10.25%。值不值？

**不能简单比较利率！** 要计算额外借的那 5%（从 80 升到 85）的**真实成本**：

```
每年利息差额 = 85 × 10.25% - 80 × 10%
            = 8.7125 - 8.0000
            = 0.7125

额外借款额 = 85 - 80 = 5

增量借贷成本 = 0.7125 / 5 = 14.25%
```

**结论：** 多借这 5 英镑，实际年利率是 14.25%——远高于表面的 10.25%！如果无杠杆 BTIRR 只有 12%，这 14.25% 的增量成本就划不来。

> **类比**：信用卡分期看起来利率不高，但"手续费摊到额外借款上"之后，实际年化成本可以高达 20%+。

## 7. 杠杆与收益率分解 Leverage, Yield & Growth

### 总回报的分解

```
总回报率 r = 收益率 y（Income Yield）+ 增值率 g（Capital Growth）
收益率 y = NOI / V（=Cap Rate）
增值率 g = 资产价值年增长率
```

### WACC 应用于收益率分量

```
整体收益率（Property Yield）：
yP = LTV × yD + (1 - LTV) × yE

整理得股权收益率（Cash-on-Cash Yield，现金回报率）：
yE = (yP - LTV × yD) / (1 - LTV)
```

**理解 yD（债务收益率）：** 对纯利息贷款（IO），yD = 利率（因为每年只付利息，没有本金偿还/增值）

### WACC 应用于增值率分量

```
整体增值率：
gP = LTV × gD + (1 - LTV) × gE

整理得股权增值率：
gE = (gP - LTV × gD) / (1 - LTV)
```

**gD（债务增值率）= 贷款价值的年变化率**：
- 纯利息（IO）贷款：gD = 0（本金不变，债务价值不变）
- 摊销贷款（CPM）：gD < 0（余额随摊销下降，债务价值"增加" ——等效于股权增加）

## 8. 日本房地产案例 Japanese Property Case Study

**情景（2006年）：** 购入 £100M 日本物业，初始 Cap Rate = 7%，净租金以 -2%/年递减（供大于求）

**假设：**
- LTV = 85%（借 £85M），利率 4%（极低！）
- 纯利息贷款，每年利息 = 85 × 4% = £3.4M

**现金收益率计算（Cash-on-Cash Yield）：**
```
yE = (yP - LTV × yD) / (1 - LTV)
   = (7% - 0.85 × 4%) / (1 - 0.85)
   = (7% - 3.4%) / 0.15
   = 3.6% / 0.15
   = 24%（2006年，相当吸引人！）
```

**但资本增值（gP）为 -2%/年（资产价值下跌）：**
```
gE = (gP - LTV × gD) / (1 - LTV)
   = (-2% - 0.85 × 0%) / 0.15
   = -2% / 0.15
   = -13.3% 每年（股权资本损失！）

年化股权资本利得 ≈ -16.6%（考虑复利效应）
```

**结论：** 现金回报率看起来 24%（诱人），但资产价值每年损失 16.6%。合计股权总回报率约 7–8%，且风险极高——市场若更差或 Cap Rate 轻微上升，将是灾难性损失。

**教训：**
- 高杠杆 + 下跌市场 = 极度放大的损失
- 现金收益率（Cash-on-Cash Yield）只是回报的一部分；资本增值同等重要
- "正杠杆"对收益率有效（yP > yD），但"负杠杆"对增值（gP < 0）会被杠杆放大

## 9. 杠杆与风险 Leverage & Risk ⭐

### 数值案例对比

**设定：** 总成本 1，持有 5 年，折现率 10%，三种情景

| 情景 | NOI₁ | 增长率 g | 售价 | 概率 |
|------|------|---------|------|------|
| 悲观 | 0.10 | -2% | 0.9 | 20% |
| 最可能 | 0.11 | 0% | 1.1 | 50% |
| 乐观 | 0.12 | +5% | 1.3 | 30% |

**无杠杆情景（Unlevered）：**

| 情景 | 估计 BTIRRP | 偏差 | 概率 | 加权方差 |
|------|------------|------|------|---------|
| 悲观 | 7.93% | -5.13% | 20% | 5.26 |
| 最可能 | 12.56% | -0.50% | 50% | 0.12 |
| 乐观 | 17.31% | +4.25% | 30% | 5.42 |

```
E(BTIRR) = 0.2×7.93 + 0.5×12.56 + 0.3×17.31 = 13.06%
Variance = 5.26 + 0.12 + 5.42 = 10.81
SD = √10.81 = 3.29%
```

**有杠杆情景（90% LTV，CPM贷款，10%利率，15年期）：**

| 情景 | 估计 BTIRRE | 概率 | 加权方差 |
|------|------------|------|---------|
| 悲观 | **-6.03%** | 20% | 198.39 |
| 最可能 | 24.96% | 50% | 0.13 |
| 乐观 | 47.30% | 30% | 143.09 |

```
E(BTIRRE) = 0.2×(-6.03) + 0.5×24.96 + 0.3×47.30 = 25.46%
Variance = 198.39 + 0.13 + 143.09 = 341.61
SD = √341.61 = 18.48%
```

**对比结论：**

| | 无杠杆 | 90% 杠杆 | 倍数 |
|--|--------|---------|------|
| E(IRR) | 13.06% | 25.46% | ≈2倍 |
| SD | 3.29% | 18.48% | ≈5.6倍 |
| 悲观情景 | 7.93% | **-6.03%** | — |

→ **预期回报翻倍，但风险放大了 5.6 倍！悲观情景下，有杠杆投资者亏损。**

### 杠杆与风险的数学关系

```
回顾：r = rf + RP（总回报 = 无风险利率 + 风险溢价）

对 WACC 公式应用于风险溢价：
RPP = LTV × RPD + (1 - LTV) × RPE

整理得：
RPE = RPD + LR × (RPP - RPD)

若债务无风险（RPD = 0）：
RPE = LR × RPP

即：股权风险溢价 ∝ 杠杆率（成正比！）
```

**核心结论：**
- 杠杆率增加 → 股权风险溢价等比例增加
- **没有免费午餐（No Free Lunch）**：高杠杆带来高回报，同时带来高风险
- 问问自己：更高的预期回报是否与更高的风险**相称**？

## 考试高频考点 Exam Focus Points

> **历年真题分析 Based on Past Exams 2013–2025**

1. **WACC 公式 + 杠杆放大效应** ★★★★★ — 2013, 2014, 2015, 2017, 2018, 2019, 2020, 2021, 2022, 2025
   - `rE = rP + (rP - rD) × D/E`
   - 注意：WACC 是**近似值**（假设 D/E 不变），全 DCF 才是精确解

2. **杠杆 IRR 计算（全 DCF 方法）** ★★★★★ — **几乎每年**（2013–2025）
   - IO 贷款：CF 中减去利息，期末减去贷款本金（balloon）
   - CPM 贷款：每期减去 PMT，期末减去剩余余额（更复杂）

3. **正/负杠杆条件 Positive/Negative Leverage** ★★★★ — 2013, 2015, 2017, 2018, 2025
   - rP > rD → 正杠杆（杠杆放大收益）
   - rP < rD → 负杠杆（杠杆放大损失，如 2013 年租金下降情景 IRR = -18.6%）

4. **情景分析 E(IRR)、Var、SD** ★★★★ — 2013, 2014, 2015, 2016, 2018
   - 两情景：E(IRR) = 50%×IRR₁ + 50%×IRR₂；SD 计算后比较有/无杠杆下的风险放大

5. **Cash-on-Cash Yield + Growth Yield** ★★★ — 2014, 2015, 2018
   - `yE = (yP - LTV×yD) / (1-LTV)`；增值收益率用初/末股权价值年化
   - 2015 考了 95% LTV 下资产价值低于贷款额（负股权）的极端情景

6. **税后 IRR After-Tax Analysis** ★★★ — 2013, 2014, 2016, 2018, 2019
   - 建筑折旧（直线法）抵税；利息税盾；资本利得税
   - 2019 有完整税后 IRR 推导题，含折旧、利息扣除、含资本利得

7. **BEIR 盈亏平衡利率** ★★ — 明确出现于 2013
   - BEIR = 无杠杆 IRR；在 BEIR 借款则杠杆无效；低于 BEIR 才有利
