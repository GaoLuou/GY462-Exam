## 1. 项目开发概述 What is Project Development?

传统投资 = 买入 → 轻改造 → 出租 → 售出

项目开发 = 买地 → 规划 → 建设 → 出租 → 管理 → 售出

```
开发 = 土地收购 + 规划/许可 + 建设融资 + 出租管理 + 退出
```

---

## 2. 规划与许可 Planning & Permitting

### 关键规划概念 Key Planning Concepts
- **Setbacks & Right-of-Way**：建筑退线要求
- **FAR（Floor-to-Area Ratio）容积率**：可建总面积/土地面积
- **Height Restrictions**：限高
- **Inclusionary Zoning**：强制配建保障房比例
- **Loss Factor 得房率**：可出租面积/总建筑面积

> 英国规划由地方政府自由裁量，差异很大！

---

## 3. 开发时间线与风险 Development Timeline & Risk

| 阶段 | 风险等级 |
|------|---------|
| 规划阶段 | 高（审批不确定） |
| 建设阶段 | 最高峰（施工风险、成本超支） |
| 完工/出租阶段 | 高（市场风险） |
| 稳定运营 | 低 |

### 现金流特征 Cash Flow Pattern
```
初期（建设）：大量负现金流（硬成本 + 软成本）
后期（出租稳定后）：正现金流
关键：用未来正现金流弥补初期负现金流
```

**硬成本 Hard Costs**（S 形曲线）：建筑材料、施工费用  
**软成本 Soft Costs**：贷款利息（负摊销 Negative Amortization）、项目管理、顾问费等

---

## 4. 建设贷款 Construction Loan

- 由专业建设贷款机构提供（了解市场、监控进度）
- **逐步放款**：按已完成工程价值按月拨付
- 原则：任意时点放款额 ≤ 当时物业价值（抵押品价值）
- 利率高于永久贷款（风险更大）

---

## 5. 永久贷款 Permanent Loan

- 建设完成后接替建设贷款
- 申请材料复杂：市场分析、政府文件、建筑成本明细、建设贷款结构
- **要约有效条件 Contingencies（常见）：**
  - 最长建设期限
  - 最低出租率（minimum rent-up）
  - 设计/材料变更审批
  - 竣工日期
- 永久贷款方通常为保险公司、养老金等机构

> **流程**：开发商先拿建设贷款 → 竣工达标 → 永久贷款偿还建设贷款并继续融资

## 6. "Build at 10, Sell at 8" 法则 ⭐

核心思路：开发项目的可行性快速判断

```
"Build at 10"：预期稳定 NOI / 总开发成本 = 10%
              (类似 Going-in Cap Rate)

"Sell at 8"：预期稳定 NOI / 出售价格 = 8%
             (类似 Going-out Cap Rate)
```

**含义**：若能以 10% 回报率开发，以 8% 资本化率出售，则：
```
利润 = 出售价格 - 开发成本
     = NOI/8% - NOI/10%
     = NOI × (1/8% - 1/10%)
     = NOI × 2.5  →  利润率约 25%
```

### 实例计算 Muk Office Plaza
| 项目 | 数值 |
|------|------|
| 土地成本 Land | £30/gross sq ft |
| 建设硬成本 Hard Costs | £90/gross sq ft |
| 软成本 Soft Costs | £30/gross sq ft |
| **总成本 Total** | **£150/gross sq ft** |
| 市场租金 Rent | £30/leasable sq ft |
| 运营成本 Op. Cost | £10/gross sq ft |
| 占用率 Occupancy | 95% |
| 得房率 Loss Factor | 30%（leasable = 70% of gross） |

**稳定 NOI 计算：**
```
每 gross sq ft NOI = £30 × 95% × 70% - £10 = £9.95
去入资本化率 = 9.95 / 150 = 6.6%  < 10% → 开发不合算
需要租金 = £37.59/leasable sq ft（比当前高 25%）
```

## 7. 预出租策略 Pre-Letting Strategy

- 提前锁定租户 → 降低未来出租不确定性
- 代价：若市场比预期好，则放弃了上行收益
- 开发商根据风险偏好决策

## 考试高频考点 Exam Focus Points

> **历年真题分析 Based on Past Exams 2013–2025**

> ⚠️ **考试趋势**：开发融资为近年新增重点（2022–2025），早期考题（2013–2021）几乎不考

1. **开发比直接买楼风险更高的原因** ★★★ — 2024, 2025
   - 规划审批不确定性、施工超支、完工延误、出租市场风险、建设期零收入
   - 2025 明确要求讨论"管理建筑商"的风险与机会

2. **建设贷款 vs 永久贷款 Construction vs Permanent Loan** ★★★ — 2022, 2024
   - 建设贷款：逐步放款（按已完成价值）、高利率、短期
   - 永久贷款：建成后接替建设贷款、需满足最低出租率等条件

3. **"Build at 10, Sell at 8" 法则计算** ★★★ — 2024（明确考察）
   - `利润 = NOI/Cap_out - NOI/Cap_in`；快速判断开发可行性
   - 反向计算：若利润率已知，反算所需租金水平

4. **合同类型 Turnkey / Forward Fund / Forward Commit** ★★★ — 2025
   - Turnkey：开发商承担全部风险直至交付；投资者最安全但回报较低
   - Forward Fund：投资者按阶段出资，分担建设风险
   - Forward Commit：投资者承诺完工后购买，开发商承担建设风险

5. **建设期 IO 贷款计算** ★★ — 2022（含建设期月供计算）
   - 建设期零 NOI + 主贷款利息 + 建设贷款利息；第 6 期加还建设贷款本金
