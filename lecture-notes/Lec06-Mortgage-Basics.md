# Lecture 6 — 抵押贷款基础 Mortgage Basics

## 核心主题 Key Topics
固定利率抵押贷款类型、月度复利、余额计算、实际借款成本、折扣点

## 1. 为什么要学抵押贷款？

在前几讲，我们假设贷款存在（如 80% LTV 的 IO 贷款），并用它计算 IRR。但我们没有深究贷款本身是怎么运作的。理解贷款机制至关重要，因为：

- **贷款利息直接影响现金流**：月供越高，投资者每年可留存的现金越少
- **贷款结构影响风险**：固定利率 vs 浮动利率，对利率变化的敏感度完全不同
- **实际成本 ≠ 名义成本**：折扣点、手续费等使实际借贷成本高于合同利率

## 2. 复利基础 Compounding

### 月度复利（Monthly Compounding）

抵押贷款通常按**月度复利**计算（Monthly Compounding），即每月的利息会加入本金，下月在更大的本金上再计利息（"利滚利"）。

**关键公式：**
```
名义年利率（Nominal Rate）= 月利率 × 12
有效年利率（EAY, Effective Annual Yield）= (1 + i/12)^12 - 1

其中 i = 名义年利率
```

**数字例：名义年利率 6%**
```
月利率 = 6% / 12 = 0.5%

第1年末：100 × (1 + 0.5%)^12 = 100 × 1.0617 = 106.17

EAY = 6.17%（> 名义的 6%，因为月度复利效应）
```

> **类比**：每月复利就像"雪球越滚越大"。即使年利率相同，月度复利比年度复利产生的利息稍多。

> **行业惯例（Convention）**：名义利率 = 有效月利率 × 12（而非按 EAY 报价）。所以当银行说"贷款利率 9.125%/年"时，实际月利率是 9.125%/12 = 0.760417%。

## 3. 固定利率贷款基础公式

### 核心变量定义

```
BAL_t  = 第 t 期末的贷款余额（Balance）
PMT_t  = 第 t 期的还款额（Payment）
AMORT_t = 第 t 期的本金偿还额（Amortization）
i      = 每期利率（月利率 = 年利率/12）
N      = 总期数
```

### 余额递推关系

```
BAL_t = (1 + i) × BAL_{t-1} - PMT_t

解读：
  本期余额 = 上期余额 × (1 + 利率) - 本期还款
  = 上期余额加利息 - 本期还款
```

### 还款额的构成

```
PMT_t = 本金偿还（AMORT_t）+ 利息支出（i × BAL_{t-1}）

即：每期还款 = 当期归还的本金 + 当期产生的利息（上期余额 × 利率）
```

> **理解关键**：还款额是利息和本金的混合。初期，余额大，利息多，本金还得少；后期，余额减小，利息少，本金还得多。

### 余额 = 剩余还款的现值

```
BAL_t = PMT_1/(1+i)^1 + PMT_2/(1+i)^2 + ... + PMT_N/(1+i)^(N-t)

即：任何时点的贷款余额 = 该时点之后所有未来还款的折现值（折现率=贷款利率）
```

## 4. 三种固定利率贷款类型 Types of Fixed Rate Mortgages

### 类型一：纯利息贷款 IO (Interest Only Loan)

**定义：** 每期只还利息，不还本金（AMORT = 0）

```
每期还款：PMT_t = i × BAL_{t-1} = i × BAL_0（本金不变）
贷款余额：BAL_t = BAL_0（始终等于初始贷款额）
最后一期：PMT_N = (1 + i) × BAL_0（利息 + 全部本金，一次性偿还）
```

**现金流图形：** 前 N-1 期等额小额还款，最后一期巨额"气球还款"（Balloon Payment）

**谁喜欢 IO 贷款？**
- **借款人（开发商/投资者）**：每期现金支出最低，前几年现金流更充裕；适合短期持有或建设期（等物业增值或出售再还本金）
- **贷款人（银行）**：不喜欢！因为本金迟迟不还，如果物业价值下跌，抵押品不足以覆盖全部贷款

---

### 类型二：固定摊销贷款 CAM (Constant Amortization Mortgage)

**定义：** 每期归还的本金相同（AMORT 固定），但由于余额减少，利息递减，所以总还款额也递减

```
每期摊销：AMORT_t = BAL_0 / N（固定！）
贷款余额：BAL_t = BAL_0 × (1 - t/N)（线性递减）
每期还款：PMT_t = i × BAL_{t-1} + BAL_0/N（利息递减 + 固定本金）
```

**现金流图形：** 第一期最高，此后线性递减

**为什么第一期还款比 CPM 高？**
- CAM 第 1 期：本金 = BAL_0/N + 利息 = i×BAL_0（最大）
- CPM 第 1 期：本金很少 + 利息 = i×BAL_0（本金相同！），但 CPM 总还款固定
- CAM 要求更快偿还本金，所以初期负担更重

---

### 类型三：等额还款贷款 CPM (Constant Payment Mortgage)

**定义：** 每期还款额完全相同（最常见的抵押贷款类型！）

```
每期还款：PMT_t = PMT（常数！）
贷款余额：BAL_t = (1 + i) × BAL_{t-1} - PMT（递减）
每期摊销：AMORT_t = PMT - i × BAL_{t-1}（递增，因为余额减少→利息减少→本金增多）
```

**现金流图形：**
- 总还款额：水平线（固定不变）
- 利息部分：逐年下降曲线
- 本金部分：逐年上升曲线
- 两条曲线之和 = 固定还款额（水平线）

**CPM 月供计算公式：**

```
PMT = BAL_0 × i / [1 - (1+i)^(-N)]

其中：
  BAL_0 = 初始贷款额
  i     = 月利率 = 年利率/12
  N     = 总还款期数（月）
```

## 5. 经典例题：30年 CPM

**条件：**
- 贷款类型：30年 CPM，月度复利
- 名义年利率：9.125%
- 贷款额：£120,000
- 首期还款：2026 年 4 月（从当月开始）

**月供计算：**
```
月利率 i = 9.125% / 12 = 0.760417%
总期数 N = 30 × 12 = 360

PMT = 120,000 × 0.00760417 / [1 - (1.00760417)^(-360)]
    = 120,000 × 0.00760417 / [1 - 0.06676]
    = 912.5 / 0.93324
    = £976.36/月
```

**计算器操作（TI BA II Plus）：**
```
1. 2nd [RESET] ENTER           → 清除所有变量
2. 2nd [P/Y] 12 ENTER          → 设置每年付款次数为12（月还款）
3. 2nd [QUIT]                  → 退出设置
4. 30 2nd [xP/Y] N             → 总期数 = 30×12 = 360
5. 9.125 I/Y                   → 输入年利率 9.125%
6. 120000 +/- PV               → 输入贷款额（负号表示借入）
7. 0 FV                        → FV = 0（30年后完全还清）
8. CPT PMT                     → 计算月供
9. 结果显示：976.36（月供）
```

**2028年底余额（第33期末，即2026年4月到2028年12月）：**

```
时间：2026年4月开始，到2028年12月底
期数：9个月（2026）+ 24个月（2027-2028）= 33期

计算器操作（AMORT功能）：
1. 接上面，不要 RESET
2. 2nd [AMORT]                 → 进入摊销工作表
3. ↓ 33 ENTER                 → 输入结束期 P2=33
4. ↓ 查看结果

显示：
  BAL = -117,614.86（还剩 £117,614.86 未还）
  PRN = 2,385.14（前33期共还了 £2,385.14 本金）
  INT = 29,834.74（前33期共付了 £29,834.74 利息）
```

> **感受贷款的缓慢偿还速度：** 30年贷款的前33个月，120,000只还了2,385本金——不到总额的2%！绝大多数还款是利息。这就是为什么长期贷款总利息支出惊人。

**CAM 对比（同条件）：**
```
每月固定摊销 = 120,000 / 360 = £333.33
第1期 CAM 还款 = 333.33 + 0.760417% × 120,000 = 333.33 + 912.5 = £1,245.83

注意：CAM 第1期 £1,245.83 >> CPM 的 £976.36！
所以 CAM 早期负担更重，但总利息支出更少（还本更快）。
```

## 6. 负摊销 Negative Amortization

**什么是负摊销？** 当每期还款额 < 当期应付利息时，不足部分会**加入贷款余额**，导致贷款越还越多！

```
条件：PMT_t < i × BAL_{t-1}
结果：AMORT_t = PMT_t - i × BAL_{t-1} < 0（负摊销！）
     BAL_t = BAL_{t-1} - AMORT_t > BAL_{t-1}（余额增加！）
```

**使用场景：** 建设期贷款（无收入，利息不付，自动滚入余额）

**注意：** 建设期结束时，贷款余额 > 原始贷款额。需要永久贷款偿还全部累积余额。

## 7. 实际借款成本 Effective Cost of Borrowing ⭐

**核心问题：** 银行说利率 8%，你实际付的是 8% 吗？不一定！因为还有其他费用。

### 什么是 YTM（Yield to Maturity，到期收益率）？

**YTM** = 使得"贷款金额 = 全部还款现值"的折现率，也就是贷款的 IRR（从贷款方角度）。

**若没有任何额外费用：**
```
YTM = 合同利率（完全一致）
```

**若有额外费用（如折扣点）：**
```
YTM > 合同利率（实际成本更高）
```

---

### 折扣点 Discount Points

**什么是折扣点？** 借款人在贷款时预先支付的一笔费用，相当于"预付利息"。

**计算方式：** 1 个折扣点（1 discount point）= 贷款额的 1%

**效果：**
- 借款人欠的金额：£1,000,000
- 但实际到手金额：£1,000,000 - 折扣点金额（如 1% = £10,000）= £990,000
- 借款人用 £990,000 的本金，却需要还 £1,000,000 的本金+利息 → **实际成本高于合同利率**

**为什么贷款机构收折扣点？**
1. **防范提前还款风险**：借款人可能利率下降后提前还款，银行就损失了利息收入。折扣点是对此的补偿。
2. **避免"流动借款人"**（Mobile Borrowers）：折扣点使不打算长期持有的借款人更贵 → 筛选出更稳定的借款人群体
3. **降低合同利率，帮助借款人资格达标**：有时降低名义利率（但收折扣点）使月供更低，帮助收入有限的借款人达到贷款资格门槛

**为什么借款人愿意付折扣点？**
1. 降低月供（名义利率低 → 月供低，即使一次性付了折扣点）
2. **提前还款期权价值**（Option Value）：如果将来利率大幅下降，可以提前还款并以更低利率再借，节省更多利息

---

### 折扣点实际成本计算例题

**例：** 贷款 £1,000,000，合同利率 8%/年，月还款，30年，含 1 个折扣点

**Step 1：先算月供（与无折扣点一样）**
```
N=360，I/Y=8，PV=1,000,000 → CPT PMT = 7,337.65/月
```

**Step 2：计算有折扣点时的 YTM（实际成本）**
```
借款人实际到手：1,000,000 × (1 - 1%) = 990,000（CF0）
每月还款：7,337.65（C01，频率360）

用 CF Worksheet 计算 IRR：
CF0 = -990,000
C01 = 7,337.65；F01 = 360
IRR CPT → 月 IRR = 0.6755%
年化实际成本 = 0.6755% × 12 = 8.106% > 合同 8%
```

**计算器详细操作：**
```
1. 2nd [FORMAT] 4 ENTER CE/C  → 设置4位小数
2. CF → 进入现金流工作表
3. CF0 = -990,000 ENTER       → 借款人实际到手（负号）
4. C01 = 7337.65 ENTER        → 每月还款
5. F01 = 360 ENTER            → 360个月
6. IRR CPT                    → 显示 0.6755%（月IRR）
7. 年化：0.6755% × 12 = 8.11%
```

---

### 提前还款对实际成本的影响

**直觉：** 折扣点是一次性前期费用。持有越久，这笔费用被"摊薄"得越厉害。若提前还款，等于把固定费用压缩在更短时间内，**实际成本更高**。

**例：5年后提前还款（持有60个月后还清）**

```
Step 1：计算 60 个月后的剩余余额
  用计算器 AMORT 功能，P2=60 → BAL = 950,698.25

Step 2：计算提前还款时的 YTM
  CF Worksheet：
  CF0 = -990,000
  C01 = 7,337.65；F01 = 59（前59个月）
  C02 = 7,337.65 + 950,698.25 = 958,035.90；F02 = 1（第60个月：最后一期 + 气球还款）
  IRR CPT → 月 IRR = 0.6875%
  年化实际成本 = 0.6875% × 12 = 8.25%
```

**1年后提前还款：**
```
剩余余额 = 991,646.31（几乎没有摊销！）
年化实际成本 = 9.05%（高出合同利率整整 1%！）
```

**规律总结：**
```
持有到期：实际成本 = 8.11%
5年后还：实际成本 = 8.25%
1年后还：实际成本 = 9.05%

→ 提前还款越早，实际成本越高！
```

> **如何选择？** 若另外有无折扣点贷款，利率 8.5%，你会选哪个？
> - 持有 30 年：选有折扣点的（8.11% < 8.5%）
> - 持有 1 年：选无折扣点的（8.5% < 9.05%）
> **拐点**约在 5–7 年左右。

---

### 提前还款罚金 Prepayment Penalties

**含义：** 借款人若提前还清贷款，需额外支付贷款余额的一定比例（如 3%）。

**效果：** 进一步增加提前还款的实际成本（保护贷款方避免再投资风险）。

**计算：**
```
若 5 年后还款 + 3% 罚金：
CF0 = -990,000
C01~C59 = 7,337.65
C60 = 7,337.65 + 950,698.25 + 950,698.25 × 3% = 987,748.41
IRR CPT → 月 IRR → 年化 ≈ 8.72%
```

## 8. 三种贷款类型综合比较

| | IO 贷款 | CAM 贷款 | CPM 贷款 |
|--|--------|---------|---------|
| 每期还款 | 利息固定，低 | 本金固定，总额递减 | **完全固定** |
| 本金归还 | 期末一次全还 | 每期均等偿还 | 逐期增加 |
| 早期月供 | 最低 | **最高** | 中等 |
| 总利息支出 | **最高** | 最低 | 中等 |
| 贷款方风险 | **最高**（本金久悬） | 最低 | 中等 |
| 对应利率 | 高（补偿高风险） | 低（还本快，安全） | 中等 |

## 考试高频考点 Exam Focus Points

> **历年真题分析 Based on Past Exams 2013–2025**

1. **CPM 月供计算（TI BA II Plus）** ★★★★★ — **每年** Q2（2013–2025）
   - 操作：`2nd [P/Y]=12 → N=年数×12 → I/Y=年利率 → PV=-贷款额 → CPT PMT`

2. **IO 贷款月供 + 气球还款 Balloon Payment** ★★★★★ — 几乎每年（2022, 2023, 2025 明确考）
   - 月供 = 贷款额 × 年利率/12（只还利息，不还本金）
   - 最后一期 = 月供 + 贷款全额

3. **CAM 贷款还款计划** ★★★★ — 2013, 2014, 2016, 2018, 2025
   - 固定摊销 = 贷款额/N；第 t 期还款 = 固定摊销 + 余额(t-1)×月利率
   - 必须手动计算（AMORT worksheet 不适用 CAM）

4. **DCR / Income Multiple 贷款资格判断** ★★★★ — 2013, 2014, 2018, 2022, 2023, 2024, 2025
   - DCR = 月收入 / 月供 ≥ 阈值（如 1.2 或 1.4）
   - Income Multiple = 贷款额 / 年收入 ≤ 阈值（如 4.5 或 5）
   - 2025 明确要求同时检验 DCR 和 Income Multiple 两个标准

5. **折扣点/发起费 + 实际借款成本（IRR）** ★★★★★ — 2013–2025（几乎每年）
   - `CF0 = -(贷款额 - 折扣点)`；按期支付 PMT；期末加剩余余额
   - 规律：**提前还款越早 → 实际成本越高**（固定成本分摊期变短）
   - 2025 考察持有到期（5.006%）、10 年还（5.0431%）、5 年还（5.1397%）对比

6. **AMORT 余额计算** ★★★★★ — 每年计算题（2013–2025）
   - 操作：`2nd [AMORT] → P1=1, P2=t → BAL（剩余余额）`
   - 用途：提前还款时的剩余本金；ARM 阶段转换时的新本金

7. **负摊销/部分摊销贷款** ★★★ — 2013, 2014, 2016, 2017, 2018, 2019, 2024
   - 月供 < 当期利息 → 余额增加（负摊销）；用 FV 键求到期余额

8. **IO vs CPM vs CAM 为何利率不同** ★★★★ — 2013, 2014, 2016, 2018, 2024, 2025
   - 银行视角：摊销越快/越前置，违约风险越低 → 利率越低（CAM < CPM < IO）
