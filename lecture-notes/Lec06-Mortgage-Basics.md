## 1. 月度复利 Monthly Compounding

```
名义年利率 (Nominal Rate) = 每月实际利率 × 12
有效年利率 EAY = (1 + i/12)^12 - 1

例: 名义 6% / 年
EAY = (1 + 0.06/12)^12 - 1 = 6.17%
```

---

## 2. 三种固定利率贷款类型 Types of Fixed Rate Mortgages

| 类型 | 英文 | 每期还款 | 特点 |
|------|------|---------|------|
| 纯利息贷款 | IO (Interest Only) | 只还利息 | 期末一次还本 |
| 固定还款贷款 | CAM (Constant Amortization Mortgage) | 本金等额 + 递减利息 | 早期还款多 |
| 等额还款贷款 | CPM (Constant Payment Mortgage) | 每期金额固定 | 最常见 |

---

## 3. CPM 核心公式 ⭐

### 每期还款额 Payment
```
PMT = BAL₀ × i / [1 - (1+i)^(-N)]

其中:
  BAL₀ = 初始贷款额
  i = 每期利率（月利率 = 年利率/12）
  N = 总期数（30年 = 360期）
```

### 余额公式 Remaining Balance
```
BALₜ = PV of remaining payments
     = PMT × [1 - (1+i)^(-(N-t))] / i
```

### 每期构成
```
PMTₜ = 利息部分 + 本金归还
     = i × BALₜ₋₁  +  AMORTₜ
```

### 计算器操作（TI BA II Plus）
```
2nd [P/Y] 12 ENTER     → 设置月度复利
30 2nd [xP/Y] N        → 总期数360
年利率 I/Y
贷款额 +/- PV
CPT PMT                → 得月供
```

---

## 4. 实例 Example

**条件：**
- 30 年 CPM，月度复利
- 年利率 9.125%
- 贷款额 £120,000

**月供：**
```
i = 9.125% / 12 = 0.760417%
N = 360
PMT = 120,000 × 0.760417% / [1 - (1.00760417)^(-360)]
    = £976.36 / 月
```

**2028年底余额（第33个月后）：**
```
BAL = 976.36 × [1-(1.00760417)^(-327)] / 0.00760417
    ≈ £118,000+ (大部分余额仍未还清)
```

## 5. 实际借款成本 Effective Cost of Borrowing ⭐

实际成本 ≠ 合同利率，因为存在额外费用：

### 折扣点 Discount Points
- 1 点 (1 point) = 贷款额的 1%（预付利息）
- 贷款方实际到手 = 贷款额 - 折扣点金额
- 实际成本（YTM）> 合同利率

```
实际成本计算：
CF0 = -(贷款额 - 折扣点)   ← 借款人实际到手
C01 = PMT（月供）
期数 = 360
IRR CPT → 月 YTM → ×12 得年化实际成本
```

**例：** 贷款 £1,000,000，利率 8%，1 个折扣点
```
月供 PMT = 7,337.65
到手 = 990,000（扣除 1% = 10,000）
月 YTM = 0.6755%（vs 合同 0.6667%）
年化实际成本 = 8.11%
```

### 提前还款 Early Repayment
- 若 5 年后还清 → 实际成本进一步提高
- 原因：折扣点等前期费用分摊期变短

```
提前还款的 YTM 计算：
CF0 = -990,000
C01~C59 = 7,337.65
C60 = 7,337.65 + 剩余余额 (950,698.25)
IRR CPT → 0.6875%/月 → 年化 8.25%
```
