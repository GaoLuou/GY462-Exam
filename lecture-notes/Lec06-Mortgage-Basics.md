## 1. 月度复利 Monthly Compounding

```
名义年利率 (Nominal Rate) = 每月实际利率 × 12
有效年利率 EAY = (1 + i/12)^12 - 1

例: 名义 6% / 年
EAY = (1 + 0.06/12)^12 - 1 = 6.17%
```

---

## 2. 三种固定利率贷款类型 Types of Fixed Rate Mortgages

| 类型 | 英文 | 每期还款 | 特点 |
|------|------|---------|------|
| 纯利息贷款 | IO (Interest Only) | 只还利息 | 期末一次还本 |
| 固定还款贷款 | CAM (Constant Amortization Mortgage) | 本金等额 + 递减利息 | 早期还款多 |
| 等额还款贷款 | CPM (Constant Payment Mortgage) | 每期金额固定 | 最常见 |

---

## 3. CPM 核心公式 ⭐

### 每期还款额 Payment
```
PMT = BAL₀ × i / [1 - (1+i)^(-N)]

其中:
  BAL₀ = 初始贷款额
  i = 每期利率（月利率 = 年利率/12）
  N = 总期数（30年 = 360期）
```

### 余额公式 Remaining Balance
```
BALₜ = PV of remaining payments
     = PMT × [1 - (1+i)^(-(N-t))] / i
```

### 每期构成
```
PMTₜ = 利息部分 + 本金归还
     = i × BALₜ₋₁  +  AMORTₜ
```

### 计算器操作（TI BA II Plus）
```
2nd [P/Y] 12 ENTER     → 设置月度复利
30 2nd [xP/Y] N        → 总期数360
年利率 I/Y
贷款额 +/- PV
CPT PMT                → 得月供
```

---

## 4. 实例 Example

**条件：**
- 30 年 CPM，月度复利
- 年利率 9.125%
- 贷款额 £120,000

**月供：**
```
i = 9.125% / 12 = 0.760417%
N = 360
PMT = 120,000 × 0.760417% / [1 - (1.00760417)^(-360)]
    = £976.36 / 月
```

**2028年底余额（第33个月后）：**
```
BAL = 976.36 × [1-(1.00760417)^(-327)] / 0.00760417
    ≈ £118,000+ (大部分余额仍未还清)
```

## 5. 实际借款成本 Effective Cost of Borrowing ⭐

实际成本 ≠ 合同利率，因为存在额外费用：

### 折扣点 Discount Points
- 1 点 (1 point) = 贷款额的 1%（预付利息）
- 贷款方实际到手 = 贷款额 - 折扣点金额
- 实际成本（YTM）> 合同利率

```
实际成本计算：
CF0 = -(贷款额 - 折扣点)   ← 借款人实际到手
C01 = PMT（月供）
期数 = 360
IRR CPT → 月 YTM → ×12 得年化实际成本
```

**例：** 贷款 £1,000,000，利率 8%，1 个折扣点
```
月供 PMT = 7,337.65
到手 = 990,000（扣除 1% = 10,000）
月 YTM = 0.6755%（vs 合同 0.6667%）
年化实际成本 = 8.11%
```

### 提前还款 Early Repayment
- 若 5 年后还清 → 实际成本进一步提高
- 原因：折扣点等前期费用分摊期变短

```
提前还款的 YTM 计算：
CF0 = -990,000
C01~C59 = 7,337.65
C60 = 7,337.65 + 剩余余额 (950,698.25)
IRR CPT → 0.6875%/月 → 年化 8.25%
```

## 考试高频考点 Exam Focus Points

> **历年真题分析 Based on Past Exams 2013–2025**

1. **CPM 月供计算（TI BA II Plus）** ★★★★★ — **每年** Q2（2013–2025）
   - 操作：`2nd [P/Y]=12 → N=年数×12 → I/Y=年利率 → PV=-贷款额 → CPT PMT`

2. **IO 贷款月供 + 气球还款 Balloon Payment** ★★★★★ — 几乎每年（2022, 2023, 2025 明确考）
   - 月供 = 贷款额 × 年利率/12（只还利息，不还本金）
   - 最后一期 = 月供 + 贷款全额

3. **CAM 贷款还款计划** ★★★★ — 2013, 2014, 2016, 2018, 2025
   - 固定摊销 = 贷款额/N；第 t 期还款 = 固定摊销 + 余额(t-1)×月利率
   - 必须手动计算（AMORT worksheet 不适用 CAM）

4. **DCR / Income Multiple 贷款资格判断** ★★★★ — 2013, 2014, 2018, 2022, 2023, 2024, 2025
   - DCR = 月收入 / 月供 ≥ 阈值（如 1.2 或 1.4）
   - Income Multiple = 贷款额 / 年收入 ≤ 阈值（如 4.5 或 5）
   - 2025 明确要求同时检验 DCR 和 Income Multiple 两个标准

5. **折扣点/发起费 + 实际借款成本（IRR）** ★★★★★ — 2013–2025（几乎每年）
   - `CF0 = -(贷款额 - 折扣点)`；按期支付 PMT；期末加剩余余额
   - 规律：**提前还款越早 → 实际成本越高**（固定成本分摊期变短）
   - 2025 考察持有到期（5.006%）、10 年还（5.0431%）、5 年还（5.1397%）对比

6. **AMORT 余额计算** ★★★★★ — 每年计算题（2013–2025）
   - 操作：`2nd [AMORT] → P1=1, P2=t → BAL（剩余余额）`
   - 用途：提前还款时的剩余本金；ARM 阶段转换时的新本金

7. **负摊销/部分摊销贷款** ★★★ — 2013, 2014, 2016, 2017, 2018, 2019, 2024
   - 月供 < 当期利息 → 余额增加（负摊销）；用 FV 键求到期余额

8. **IO vs CPM vs CAM 为何利率不同** ★★★★ — 2013, 2014, 2016, 2018, 2024, 2025
   - 银行视角：摊销越快/越前置，违约风险越低 → 利率越低（CAM < CPM < IO）
