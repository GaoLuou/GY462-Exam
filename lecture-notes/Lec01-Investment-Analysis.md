## 1. 房地产与其他金融资产的区别 Characteristics of Real Estate vs. Other Assets

**为什么要单独研究房地产金融？** 因为房地产与股票、债券等金融资产有根本性差异，这些差异决定了它的独特风险与机会。

与其他金融资产的区别 Differences from other financial assets：

- **不可分割性 Indivisibility（大宗交易）**：你不能只买一栋楼的"一半"。买楼往往需要数百万甚至数十亿资金，难以拆分，迫使投资者使用杠杆（借贷）。
- **异质性 Heterogeneity（每套物业独一无二）**：相邻两栋楼、朝向不同、楼龄不同，定价完全不同。这使得估值特别依赖专业判断，而非市场统一报价。
- **流动性差 Illiquidity（难以快速变现）**：卖楼通常需要数月，交易成本高（印花税、中介费、法律费），且多为私下谈判完成（private trading），无公开透明市场。
- **高交易成本 High Transaction Costs**：买卖一套商业物业，交易费用可达成交价的 5–10%，而股票仅约 0.1%。

> **类比**：股票像超市里的商品，标价清晰、随时可买卖；房产像一件孤品古董，需要议价、等待、鉴定，卖掉需要几个月。

**这些特征如何影响风险与机会？**
- 流动性差 → 投资者要求更高的风险溢价（流动性溢价 Liquidity Premium）
- 不可分割性 → 必须借贷，引入杠杆风险
- 异质性 → 信息不对称，专业投资者可凭经验获得超额回报

## 2. 收益性物业类型 Income-Producing Property Types

**什么是收益性物业？** 能产生租金收入的物业（Income-Producing Properties）。与自住物业不同，它的价值核心是：**未来租金现金流的折现值**。

| 类型 Type | 特点 Notes |
|-----------|-----------|
| 办公 Office | 核心 CBD 高层 vs 郊区小楼；大单一租户 vs 多租户（流动性更好） |
| 零售 Retail | 购物中心（Westfield 类）vs 商业街（Oxford Street）；"锚定租户"（如超市）极重要 |
| 住宅 Residential/Multifamily | 高层公寓楼、小户型公寓；租户分散，单笔违约影响小 |
| 工业/仓储 Industrial/Warehouse | 靠近高速公路出入口是关键（亚马逊物流仓为代表）；电商崛起推高需求 |
| 酒店 Hotel | 极难估值（Cap Rate 差异巨大），因为租期是"按天"，波动性极高，放在灰色区域 |

**三大影响维度（考试常考！）：Location（位置）、Structure（结构）、Tenants（租户）**

- **位置**：决定需求基础。写字楼要交通枢纽，仓库要物流便捷，购物中心要人流量大。
- **结构**：楼龄、设计、技术标准（办公楼能否支持高密度网络设备？仓库层高够不够？）
- **租户**：谁在租？信用评级好的大企业租户（如政府、Fortune 500）违约风险极低，大幅降低折现率。

## 3. 房地产市场供求影响因素 Demand & Supply

**需求侧**（决定多少人想租）：
- 就业率和收入分布（企业繁荣 → 扩张办公空间）
- 集聚效应（金融公司都想在金融城，形成正反馈）
- 人口年龄结构（年轻人多 → 住宅需求旺）

**供应侧**（决定市场有多少空间可租）：
- 当前空置率（Vacancy Rate）：空置越高 → 供应过剩 → 租金承压
- 土地和建造成本（成本高 → 新供应受抑制）
- 利率和融资可及性（借贷成本高 → 开发减少）
- 现有楼宇老化程度

## 4. DCF 估值法 Discounted Cash Flow Valuation ⭐

**核心思想**：一栋楼今天值多少钱？等于它未来所有租金和售价，折回到今天的"现值"之和。

### 为什么要折现？
因为"今天的 100 英镑"比"明年的 100 英镑"更值钱：
1. 通胀侵蚀购买力
2. 今天的钱可以再投资
3. 未来的钱存在风险（可能拿不到）

### DCF 三步骤
1. **预测未来现金流**（Forecast Future Cash Flows）：估算每年租金净收入 NOI 和最终售价
2. **确定折现率**（Ascertain Required Rate of Return）：反映投资风险的"门槛回报率"
3. **折现至现值**（Discount to Present Value）：把所有未来现金流折回今天

```
PV = CF₁/(1+r)¹ + CF₂/(1+r)² + ... + CFₙ/(1+r)ⁿ + 售价ₙ/(1+r)ⁿ

其中：
  PV = 现值（今天物业的价值）
  CFₜ = 第 t 年的现金流（租金净收入）
  r   = 折现率（必要回报率）
  n   = 持有年限
```

### 经典例题：单一租户办公楼（Single Tenant Office Building）

**条件：**
- 6 年租约：第 1–3 年租金每年 £1.0M，第 4–6 年租金每年 £1.5M（租金上调，Step-up）
- 第 6 年末出售，预计售价 £15M
- 折现率 r = 10%

**计算思路（三段分别折现，然后相加）：**
```
第一段：第 1–3 年，年 CF = 1.0
PV₁ = 1.0/1.1¹ + 1.0/1.1² + 1.0/1.1³

使用年金公式（TI BA II Plus 计算器）：
  N=3, I/Y=10, PMT=1.0, FV=0 → CPT PV = 2.487

第二段：第 4–6 年，年 CF = 1.5，但需先折到第 3 年末再折回 t=0
  N=3, I/Y=10, PMT=1.5, FV=0 → CPT PV(at t=3) = 3.730
  再折 3 年到 t=0：PV₂ = 3.730 / 1.1³ = 2.803

第三段：售价 15.0（加上第 6 年租金 1.5 = 16.5），在第 6 年末
  PV₃ = 16.5 / 1.1⁶ = 8.467

总价值 = 2.487 + 2.803 + 8.467 = 13.757
```

**答案：** 该办公楼今天价值约 **£13.757M**

> **考试技巧**：上述分三段是一种方法。也可以用计算器 CF Worksheet 逐年输入现金流，然后用 NPV 键直接计算。

## 5. 折现率 Discount Rate — 如何选择？

### 折现率的构成
```
r = r_f + RP

其中：
  r_f = 无风险利率（Risk-Free Rate）= 国债收益率（Treasury Bill Yield）
        代表货币的时间价值（Time Value of Money）
  RP  = 风险溢价（Risk Premium）
        反映投资者为承担额外风险要求的额外回报
```

> **关键理解**：RP 不是"绝对风险"，而是该投资对**整体投资组合**的边际风险贡献。一栋与股市高度相关的办公楼，风险溢价比一栋政府租约的楼高得多。

### 折现率应反映什么？
- 项目的**风险特征**（Risk Characteristics）：类似物业的历史波动性
- **资本机会成本**（Opportunity Cost of Capital）：如果不买这栋楼，最好的替代投资能赚多少？
- **同类投资的市场惯例**：行业观察到的、对类似物业使用的折现率

### 折现率的来源（实际操作）
- 市场上已成交的类似物业的 IRR
- 行业报告（如 JLL、CBRE 发布的市场数据）
- 加成国债利率 + 物业风险溢价

## 6. 租约内 vs 租约间折现率 Intra vs Inter-Lease Discount Rate ⭐⭐

这是 Lecture 1 最高频的考点，理解它需要先明白：**不同阶段的现金流风险不同，应该用不同的折现率！**

### 核心区别

| | 租约内 Intra-Lease（合同期内） | 租约间 Inter-Lease（合同到期后） |
|--|-------------------------------|--------------------------------|
| 含义 | 已签合同、有法律约束力的现金流 | 旧租约到期后的下一段，尚未签约 |
| 主要风险 | **租户违约风险**（Tenant Default Risk）：租户可能破产，但合同已签 | **市场风险**（Market Risk）：未来经济好不好？租金涨跌？能不能找到新租户？ |
| 折现率 | **低**（如 8%）——因为合同锁定，风险主要是违约概率 | **高**（如 15%）——未来不确定性大得多 |

> **类比**：租约内就像你手里拿着国债利息承诺，风险低；租约间就像你等着利率不确定的投资产品，风险高。

### 五步法计算（必须掌握！）

**例题设定：** 在 2000 年估价某物业
- 第一租约（已签）：2001–2003 年每年 £1.0M，2004–2006 年每年 £1.5M，折现率 8%（intra-lease）
- 第二租约（预期）：2007–2010 年每年 £2.0M，折现率 8%（intra-lease，一旦签约后）
- 2010 年末出售：预计 £20M
- 折现从 2000 年开始的"不确定性"部分（第二租约 + 售价）：用 15%（inter-lease）

**Step 1：计算第一段租约在 2000 年的 PV（用 intra-lease 折现率 8%）**
```
CF = 1.0（2001–2003），1.5（2004–2006）

计算器：
  N=3, I/Y=8, PMT=1.0, FV=0 → PV(t=2003) = 2.577
  N=3, I/Y=8, PMT=1.5, FV=0 → PV(at t=2003) = 3.869；折回 2000 = 3.069

PV(第一段租约, 2000) = 2.577 + 3.069 = 5.646
```

**Step 2：计算第二段租约在 2006 年末的 PV（用 intra-lease 折现率 8%，假设 2006 年时已签约）**
```
CF = 2.0（2007–2010）
N=4, I/Y=8, PMT=2.0, FV=0 → PV(t=2006) = 6.624
```

**Step 3：将 Step 2 的结果（在 2006 年的 6.624）折回 2000 年（用 inter-lease 折现率 15%）**
```
PV(第二段租约, 2000) = 6.624 / (1.15)⁶ = 2.864
```

**Step 4：计算 2010 年售价在 2000 年的 PV（用 inter-lease 折现率 15%）**
```
PV(售价, 2000) = 20 / (1.15)¹⁰ = 4.944
```

**Step 5：汇总**
```
物业总价值（2000年）= 5.646 + 2.864 + 4.944 = 13.454
```

### 与简单法（全程用单一折现率）的对比

如果对 10 年全程使用同一折现率（blended rate，大约 10%），结果约为 £13.058。

**差额：13.454 − 13.058 = £0.396M**

> **这就是精明投资者的超额收益！** 准确理解哪部分现金流风险低（用低折现率），哪部分风险高（用高折现率），可以发现市场定价中的机会，额外获利约 £40 万。

### 实践中的复杂性

- 一栋楼可能有**多个租户**，每个租户的租约到期时间不同，需要逐一匹配
- 运营费用也有不确定性（不完全固定）
- 实践中，很多投资者使用**混合折现率（blended rate）或 IRR** 简化计算，我们下节课详细讨论

## 7. 其他估值方法 Alternative Valuation Methods

DCF 法并非唯一的物业估值方法，还有：

| 方法 Method | 核心逻辑 | 适用场景 |
|------------|---------|--------|
| 销售比较法 Sales Comparison Approach | 参考近期类似物业成交价，调整差异（面积、楼层、装修等） | 市场活跃、可比交易多时最可靠 |
| 建设成本法 Construction Cost Approach | 估算重建成本（土地＋建安＋折旧减值） | 适用于特殊用途物业（如学校、工厂），市场可比数据少 |

**三种方法应互相印证：** 若 DCF 法、销售比较法和建设成本法结果接近，估值较可信。若差异大，说明模型假设有问题，需要重新审视。

## 8. 主要风险与机会回顾 Risks & Opportunities

### 主要风险 Downside Risks
1. **运营成本**（Operating Cost Risk）：维修、管理费超预算
2. **自然灾害**（Natural Disaster）：洪水、地震
3. **空置**（Vacancy Risk）：租户不续约，物业空置期无收入 ← **通常最重要的运营风险**
4. **租赁（违约）风险**（Leasing/Default Risk）：租户破产，无法按合同付租
5. **流动性风险**（Liquidity Risk）：需要卖楼时找不到买家，或折价严重

### 主要机会 Opportunities
1. **运营成本节约**：比预期更低的维修、能耗
2. **终值（资产增值）**（Terminal Value）：卖楼时溢价出售
3. **租金增长**（Rental Growth）：经济好转时，租金上涨超预期 ← **通常最重要的机会来源**

> **DCF 角度理解**：NOI（净营业收入）代表"机会"，折现率代表"风险"。两者共同决定物业价值。

1. **RE 主要风险 Main Risks** ★★★★★ — 出现于**每年** Theory Q1（2013–2025）
   - 空置、流动性差、经营成本、租赁违约、自然灾害、信息不对称
   - 必须区分"房地产特有风险"vs"其他金融资产共有风险"

2. **RE 主要机会 Opportunities** ★★★★ — 2013, 2014, 2015, 2018, 2022, 2023, 2025
   - 租金增值、资产升值、运营管理协同效应（租金价值 > 售价重要）

3. **流动性差/不可分割性 Illiquidity / Indivisibility** ★★★★ — 2019, 2020, 2023, 2025
   - 三大特征：Indivisibility（大宗交易）、Illiquidity（难变现）、Large-lot size
   - 这些特征使杠杆成为不可避免的选择，进一步放大风险

4. **DCF 方法如何体现风险与机会** ★★★★ — 2013, 2014, 2019, 2021, 2025
   - 分子（NOI）= 机会；分母（折现率）= 风险
   - 同一资产不同租约段用不同折现率（Intra vs Inter-Lease）

5. **Intra/Inter-Lease 折现率五步法** ★★ — 明确计算出现于 2016
   - Intra-lease（签约期内）：低折现率（租户违约风险低）
   - Inter-lease（租约间/未来重新出租）：高折现率（市场风险高）

6. **物业类型特征 Property Types（Office/Retail/Industrial）** ★★ — 2013, 2014, 2015
   - 位置、结构、租户特征对 Cap Rate 和风险的影响
