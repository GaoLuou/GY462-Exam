## 第一部分：计算器布局与基本操作
## Part 1: Calculator Layout & Basic Operations

### 关键按键说明 Key Buttons

| 按键 Key | 功能 Function |
|---------|-------------|
| `EXE` | 执行/等号 Execute (like = ) |
| `AC` | 全清 All Clear |
| `DEL` / `◀` | 删除上一字符 Delete |
| `SHIFT` | 第二功能（蓝色标注）Second function |
| `SHIFT` + `AC` | 关机 OFF |
| `ANS` | 上一步结果 Last answer |
| `x10ˣ` | 输入 ×10ⁿ（科学计数法）|
| `FORMAT` | 切换小数/分数/科学计数显示 |
| `VARIABLE` (z̄x) | 变量存储与调用 |
| `FUNCTION` f(x) | 函数计算 |
| `CATALOG` | 所有函数目录 |
| `TOOLS` (···) | 工具菜单（角度/小数设置等）|
| `HOME` | 返回主菜单 |
| `SETTINGS` | 系统设置 |

### 模式切换 Mode Selection

按 `HOME` 进入主菜单，选择：
- **Calculate**：标准计算模式（★ GY462 主要使用这个）
- **Statistics**：统计模式（用于计算 E(IRR)、方差、标准差）
- **Table**：表格模式（可批量计算不同期数的现值）
- **Ratio / Math Box**：不用于本课程

## 第二部分：内存与变量使用
## Part 2: Memory & Variable Storage

### 存储变量 Store to Variable

```
计算结果后 → SHIFT → RCL → 选字母 (A/B/C/D/E/F)
或者：先输入数值 → VARIABLE → 选 Store → 选字母
```

**例：将月利率 0.004083 存入变量 A：**
```
4.9 ÷ 100 ÷ 12 EXE    →  得 0.004083...
SHIFT → RCL → A        →  存入 A
```

### 调用变量 Recall Variable

```
VARIABLE → 字母选择 → EXE
或直接在算式中输入变量字母（按 VARIABLE 键选择）
```

### ANS 键用法 Using ANS

- 每次按 `EXE` 后，结果自动存入 `ANS`
- 下一步计算可直接用 `ANS` 引用上一结果
- 用于**链式计算**非常高效

```
例：先算 (1+i)^(-N)，再用 ANS 计算分母
(1 + A)^(-300) EXE     →  结果存入 ANS
1 - ANS EXE            →  分母
```

## 第三部分：核心金融公式与操作步骤
## Part 3: Core Financial Formulas & Keystrokes

> ⚠️ 以下公式是 TI BA II Plus 内置功能在 Casio 上的**手动等价方法**

---

### 3.1 月利率计算 Monthly Interest Rate

```
i_monthly = 年利率 / 12 / 100

操作：
年利率数值 ÷ 12 ÷ 100 EXE
→ 建议存入变量 A（后续复用）
```

**例：年利率 4.9%**
```
4.9 ÷ 12 ÷ 100 EXE  →  0.00408333...
SHIFT RCL A           →  存入 A
```

---

### 3.2 CPM 月供公式 Monthly Payment (Constant Payment Mortgage)

$$PMT = \frac{PV \times i}{1 - (1+i)^{-N}}$$

**操作步骤（以 £750,000，25年，4.9% 为例）：**

```
步骤1：月利率存入 A
4.9 ÷ 100 ÷ 12 EXE  →  0.004083...
SHIFT RCL A

步骤2：计算 (1+i)^(-N)，N=25×12=300
( 1 + A ) ^ ( (-) 300 ) EXE  →  约 0.29415...
SHIFT RCL B                    →  存入 B

步骤3：计算分母 1-(1+i)^(-N)
1 - B EXE                     →  约 0.70585...
SHIFT RCL C                    →  存入 C

步骤4：计算 PMT
750000 × A ÷ C EXE            →  ≈ £4,340.84
```

> 💡 负号输入：按 `(-)` 键（在 3 键右下方），不是减号 `-`

---

### 3.3 IO 贷款月供 Interest Only Monthly Payment

$$PMT_{IO} = PV \times i$$

```
操作（贷款 £750,000，年利率 5.5%）：
750000 × 5.5 ÷ 100 ÷ 12 EXE  →  £3,437.50
```

---

### 3.4 CAM 贷款还款计划 Constant Amortization Mortgage

```
每期固定摊销 = 贷款额 ÷ 总期数

第 t 期利息 = 余额(t-1) × 月利率
第 t 期余额(t-1) = 贷款额 - (t-1) × 固定摊销
第 t 期总还款 = 固定摊销 + 第 t 期利息
```

**例：£750,000，25年（300期），4.5%，求第1期和第12期：**

```
固定摊销：
750000 ÷ 300 EXE  →  2500（存入 A）

第1期利息：
750000 × 4.5 ÷ 100 ÷ 12 EXE  →  2812.50

第1期总还款：
2500 + 2812.5 EXE  →  £5,312.50  ✓

第12期：余额 = 750000 - 11×2500 = 722500
722500 × 4.5 ÷ 100 ÷ 12 EXE  →  2709.375
2500 + ANS EXE                →  £5,209.375  ✓
```

---

### 3.5 CPM 剩余余额公式 Outstanding Balance at Period t

$$BAL_t = PMT \times \frac{1-(1+i)^{-(N-t)}}{i}$$

**例：月供 £4,340.84，月利率 0.004083，求第60期（5年）后余额：**

```
月利率存入 A：0.004083...（已存）
月供存入 D：
4340.84 SHIFT RCL D

剩余期数 = 300 - 60 = 240

计算 (1+i)^(-240)：
( 1 + A )^( (-) 240 ) EXE  →  约 0.37716...
SHIFT RCL B

计算分子 1 - (1+i)^(-240)：
1 - B EXE  →  约 0.62284...
SHIFT RCL C

计算余额：
D × C ÷ A EXE  →  ≈ £663,286
```

> ✅ 验证：2025 年考题答案正是 £663,286.58

---

### 3.6 有效借款成本 Effective Cost of Borrowing（IRR 近似法）

TI BA II Plus 用 CF Worksheet 一键求 IRR，Casio 需要**试错法（逼近法）**。

**方法：二分法逼近 IRR**

$$NPV = CF_0 + \sum_{t=1}^{N} \frac{CF_t}{(1+r)^t} = 0$$

当 NPV = 0 时的 r 就是 IRR（月 IRR × 12 = 年化）

**操作步骤（以带折扣点的 CPM 为例）：**

**已知**：贷款 £750,000，1% 折扣点，月供 £4,340.84，持有到期（300期），求 IRR

```
实际到手 = 750000 × 99% = 742,500

思路：找 r 使得
NPV = -742500 + 4340.84 × (1-(1+r)^(-300)) / r = 0

试 r = 0.004166... （对应年化 5.0%）：
PV of payments = 4340.84 × ( 1-(1.004166)^(-300) ) / 0.004166

计算 (1.004166)^(-300)：
1.004166 ^ (-300) EXE  →  约 0.28622...

分子：1 - 0.28622 = 0.71378
PV = 4340.84 × 0.71378 ÷ 0.004166  →  约 742,800（偏高）

试 r 更小一点... 逐步逼近

```

> ⚠️ **实用建议**：对于复杂的多段 ARM + 折扣点的 IRR，试错法非常耗时。
> 建议考前记住以下规律，通过**估算法**得分：
> - 无任何费用时：IRR = 名义利率
> - 有折扣点时：IRR 比名义利率高约 `折扣点% / 持有年数`
> - 提前还款越早，IRR 越高

---

### 3.7 终值计算 Future Value (FV)

$$FV = PV \times (1+i)^N$$

```
例：£15,000 以 5% 年利率，10年后值多少？
15000 × 1.05^10 EXE  →  £24,433.42
```

---

### 3.8 现值计算 Present Value (PV)

$$PV = \frac{FV}{(1+r)^t}$$

```
例：10年后收到 £1,000,000，折现率 10%：
1000000 ÷ 1.1^10 EXE  →  £385,543.29
```

---

### 3.9 DCF / NPV 计算 Discounted Cash Flow

$$NPV = \sum_{t=0}^{N} \frac{CF_t}{(1+r)^t}$$

**例：GY462 标准 5 年持有物业**

| 年份 | 现金流 |
|-----|-------|
| 0 | -10,000,000 |
| 1–5 (每年) | 750,000 |
| 5（售价） | 12,000,000 |

```
折现率 r = 10%，年度复利

PV(第1年) = 750000 ÷ 1.1 EXE  →  SHIFT RCL A
PV(第2年) = 750000 ÷ 1.1^2 EXE →  + ANS
... 重复到第5年
PV(售价) = 12000000 ÷ 1.1^5 EXE →  + ANS
- 10000000 EXE  →  得 NPV
```

> 💡 **等比年金公式**（相同现金流多期）：  
> $PV_{annuity} = PMT \times \frac{1-(1+r)^{-N}}{r}$  
> 可用与 CPM 公式相同的按键方法

## 第四部分：典型 GY462 考题操作
## Part 4: GY462 Exam Question Walkthroughs

---

### 题型1：CPM月供 + 有效借款成本（2025年Q2b/g题型）

**题目**：贷款 £750,000，25年，月复利，年利率 4.9%，1% 发起费，持有到期，求有效借款成本

**Step 1：求月供 PMT**
```
i = 4.9 ÷ 100 ÷ 12 EXE       →  0.004083... (存A)
(1+A)^(-300) EXE               →  (存B)
1 - B EXE                      →  (存C)
750000 × A ÷ C EXE             →  PMT = £4,340.84 (存D)
```

**Step 2：确认实际到手金额**
```
750000 × 0.99 EXE  →  742,500（这是 CF0，借款人实际收到）
```

**Step 3：试错求 IRR（月 IRR）**

目标：找 r 使得 `742500 = 4340.84 × (1-(1+r)^(-300)) / r`

```
试 r = 0.004172（对应年化 5.006%）：
(1 + 0.004172)^(-300) EXE     →  约 0.2869...
1 - ANS EXE                   →  约 0.7131...
4340.84 × ANS ÷ 0.004172 EXE →  约 742,500 ✓

年化 IRR = 0.004172 × 12 = 5.006%
```

> ✅ 与 2025 考题标准答案 5.006% 一致

---

### 题型2：CAM第一期和第12期还款（2025年Q2d题型）

**题目**：贷款 £750,000，25年（300期），年利率 4.5%

```
月利率 = 4.5 ÷ 100 ÷ 12 = 0.00375
月摊销 = 750000 ÷ 300 = 2500

第1期：
利息 = 750000 × 0.00375 EXE  →  2812.5
总还款 = 2500 + 2812.5 EXE   →  £5,312.5 ✓

第12期（已还11期，余额 = 750000 - 11×2500 = 722500）：
利息 = 722500 × 0.00375 EXE  →  2709.375
总还款 = 2500 + ANS EXE      →  £5,209.375 ✓

最后一期（已还299期，余额 = 2500）：
利息 = 2500 × 0.00375 EXE   →  9.375
总还款 = 2500 + 9.375 EXE   →  £2,509.375 ✓
```

---

### 题型3：ARM 分段月供（2025年Q2i题型）

**题目**：£750,000，25年，前3年4.9%，年4-5调至4.5%，第6年起5%，求各段月供

```
=== 第1段：4.9%，25年，750000 ===
i1 = 4.9 ÷ 100 ÷ 12 EXE    →  0.004083 (存A)
(1+A)^(-300) EXE             →  (存B)
PMT1 = 750000×A÷(1-B) EXE  →  £4,340.84 (存D)

=== 求第36期（3年）后余额 ===
剩余期数 = 300 - 36 = 264
BAL36 = D × (1-(1+A)^(-264)) ÷ A EXE
具体按键：
(1+A)^(-264) EXE             →  (存E)
D × (1-E) ÷ A EXE           →  ≈ £700,533.80 (存F)

=== 第2段：4.5%，剩余22年(264期)，本金=F ===
i2 = 4.5 ÷ 100 ÷ 12 EXE    →  0.00375 (存A，覆盖)
(1+A)^(-264) EXE             →  (存B)
PMT2 = F×A÷(1-B) EXE       →  £4,184.89 (存D)

=== 求第24期（2年）后余额（从第2段起计）===
BAL24 = D × (1-(1+A)^(-240)) ÷ A EXE
→  ≈ £661,486.79 (存G)

=== 第3段：5%，剩余20年(240期)，本金=G ===
i3 = 5 ÷ 100 ÷ 12 EXE      →  0.004167 (存A)
(1+A)^(-240) EXE             →  (存B)
PMT3 = G×A÷(1-B) EXE       →  £4,365.52 (存D)
```

> ✅ 与 2025 考题标准答案一致

---

### 题型4：Pro Forma IRR（物业投资年度现金流）

**题目**：物业价格 £10,000,000，年 NOI £750,000，5年后售 £12,500,000，求 IRR

```
用试错法：目标 NPV = 0

试 r = 11%：
PV1 = 750000 ÷ 1.11 EXE      →  (存A)
PV2 = 750000 ÷ 1.11^2 EXE   →  + A (存A)
PV3 = 750000 ÷ 1.11^3 EXE   →  + A (存A)
PV4 = 750000 ÷ 1.11^4 EXE   →  + A (存A)
PV5 = (750000+12500000) ÷ 1.11^5 EXE → + A (存A)
NPV = A - 10000000 EXE        →  若约为0则IRR≈11%

或用等比年金简化（NOI 相同）：
PV_annuity = 750000 × (1-1.11^(-5)) ÷ 0.11 EXE
PV_sale = 12500000 ÷ 1.11^5 EXE
NPV = PV_annuity + PV_sale - 10000000 EXE
```

### 题型5：统计模式求 E(IRR) 和标准差（Q1情景分析）

**题目**：三种情景 IRR：13.59%（1/3）、17.68%（1/3）、10.47%（1/3），求 E(IRR) 和 SD

**使用 Statistics 模式：**

```
HOME → Statistics → 1-Variable（单变量统计）

输入数据：
13.59  EXE
17.68  EXE
10.47  EXE

然后查看结果：
x̄（mean）= E(IRR)
σx（population SD）= 标准差
```

> 💡 也可以手动计算：
> ```
> E = (13.59 + 17.68 + 10.47) ÷ 3 EXE  →  约 13.91%
> Var = ((13.59-13.91)^2 + (17.68-13.91)^2 + (10.47-13.91)^2) ÷ 3 EXE
> SD = √Var → 按 √ 键
> ```

## 第五部分：实用操作技巧
## Part 5: Practical Tips

### 技巧1：变量分配建议（考场标准化）

| 变量 | 固定用途 |
|-----|---------|
| **A** | 月利率 i |
| **B** | (1+i)^(-N)，折现因子 |
| **C** | 分母 1-(1+i)^(-N) |
| **D** | 月供 PMT |
| **E** | 阶段余额 BAL（临时） |
| **F** | 第2段本金 |
| **G** | 第3段本金 |

### 技巧2：链式计算减少输入错误

```
不要：每步重新输入数字
要用：ANS + 变量，把每步结果存起来
```

### 技巧3：幂次输入方法

```
(1+A)^(-300) 的输入顺序：
按 ( → 1 + VARIABLE→A → ) → ^ → ( → (-) → 300 → ) → EXE
```

### 技巧4：确认小数显示位数

```
FORMAT 键（或 SHIFT + FORMAT）→ 调整显示位数
建议设为 Fix 2（2位小数）看答案；
重要计算时用 Fix 4 或更多防止精度损失
```

### 技巧5：负数输入

```
负数用 (-) 键（在数字 3 右下），不是减法键 -
例：输入 -300，按 (-) 3 0 0
```

### 技巧6：常见错误排查

| 问题 | 原因 | 解决 |
|------|------|------|
| 结果与标准答案差很多 | 月利率没有除以 100 | `年利率 ÷ 100 ÷ 12` |
| (1+i)^(-N) 结果很大 | 负号用了减法键 | 用 `(-)` 而非 `-` |
| 变量 A 显示奇怪值 | 之前存了其他值 | AC 后重新存入 |
| CPM 结果明显偏高/低 | N 没有乘以 12 | 检查期数是否为月数 |

## 第六部分：关键公式速查表
## Part 6: Quick Formula Reference

```
月利率：      i = 年利率 ÷ 100 ÷ 12

CPM 月供：    PMT = PV × i ÷ (1 - (1+i)^(-N))

IO 月供：     PMT = PV × i

CAM 摊销：    每期固定摊销 = PV ÷ N
              第t期利息 = (PV - (t-1)×固定摊销) × i
              第t期总还款 = 固定摊销 + 第t期利息

剩余余额：    BAL_t = PMT × (1-(1+i)^(-(N-t))) ÷ i

终值：        FV = PV × (1+i)^N

现值：        PV = FV ÷ (1+i)^N

等比年金现值：PV = PMT × (1-(1+i)^(-N)) ÷ i

WACC：        rE = rP + (rP - rD) × D/E

Gordon 模型： V = NOI ÷ (r - g)

DCR：         月收入 ÷ 月供 ≥ DCR阈值
```
